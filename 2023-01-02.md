## 프리티어

**프리티어 설정 (es6 모듈 관련)**

```javascript
// .prettierrc.json
{
  "printWidth": 100,
  "tabWidth": 4,
  "useTabs": false,
  "semi": true,
  "singleQuote": false,
  "trailingComma": "all",
  "bracketSpacing": true,
  "arrowParens": "always",
  "proseWrap": "never",
  "endOfLine": "auto"
}
```

**프리티어 명령어**

`npx prettier --write .`

프리티어 적용하기(규칙에 따라 되지 않은 것들은 자동 변환해줌)

`npx prettier --check .`

프리티어가 최종적으로 적용되었는지 확인
